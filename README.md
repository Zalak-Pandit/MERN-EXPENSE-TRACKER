# MERN-EXPENSE-TRACKER

(pictures added)

to run this: 
1. fork this repository
2. open one cmd terminal:
       npm install
       cd FrontEnd
       npm run dev
4. open another cmd terminal:
       npm install
       cd BackEnd
       npm start

Functionalities:
1. Add and Delete Income (CRUD)
2. Add and Delete Expenses
3. Visualisation using different charts
4. Download Income and Expense in Excel format
5. Categorise Income and Expense
6. Customise Profile
7. Manage different profiles using API and MongoDB
   
This is a gist of the website
